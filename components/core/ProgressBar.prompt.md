Slim ink-outlined meter for XP, mastery %, or any stat. The fill animates its width over 0.8s.

```jsx
<ProgressBar value={90} color="sky" />
<ProgressBar value={65} color="mint" />
```

`value` is 0–100; `color` maps to the palette (`sunshine` / `coral` / `sky` / `mint` / `purple` / `ink`). Pair with a Press-Start-2P `%` label to the right for the "quest progress" pattern.
