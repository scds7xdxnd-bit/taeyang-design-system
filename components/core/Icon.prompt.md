Lucide-style stroke icon (24×24, weight 2, round caps) drawn in `currentColor` — the brand's functional icon set. Inherits color from its parent, so set `color` on the wrapper.

```jsx
<Icon name="flask" />
<Icon name="globe" size={18} />
<span style={{ color: 'var(--coral)' }}><Icon name="check" /></span>
```

Names: `globe`, `flask`, `code`, `users`, `cap`, `dashboard`, `book`, `award`, `check`, `arrowRight`, `external`. For glyphs outside this set, use Lucide at the same weight. Emoji are used separately as expressive accents — see the README's ICONOGRAPHY section.
