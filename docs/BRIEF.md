Design in the **Taeyang Han** style: a playful, game-inspired system on a **cream** (`#FFF7E6`) notebook background.

**Colors:** sunshine `#FFD859` (primary accent), coral `#FF6B6B`, sky `#6EC6FF`, mint `#7EE6A7`, purple `#B084F5`, ink `#1E1E1E` (text / borders / shadows).

**Type:** DM Sans (bold, `-0.02em` tracking) for display and headings, Inter for body, Press Start 2P for tiny UPPERCASE pixel labels and badges only — never body copy. Caveat for handwritten asides.

**Signature look:** thick 2–3px **ink borders** + **hard offset shadows with no blur** (`4px 4px 0 #1E1E1E`). On hover, lift `2px` and grow the shadow; on press, sink `1px` and shrink it — like physical stickers on a page.

**Shapes:** 8px radii (999px for pills), 8pt spacing scale, max content width 1040px.

**Patterns:** frame content as game "panels" with colored headers; "quests" with progress bars and `%` labels; status pills; badges. Coral / purple fills use white text; sunshine / sky / mint fills use ink text. Focus rings are `3px solid coral`.

**Tone:** short, clear, encouraging, level-up energy. Optional pixel-art mascot (boy + Shiba dog) and Seoul / Namsan skyline.

**Dark mode ("Night World"):** bg `#1A1917`, panels `#242220`, text `#E8E6E0`; accents mute slightly (sunshine `#D4A800`, coral `#E05555`, sky `#5AAEE0`, mint `#5DC985`, purple `#9B6FE0`); shadows flip to the light ink.
