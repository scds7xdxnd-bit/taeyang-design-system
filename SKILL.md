The heart of the brand: a *cream notebook page, inked in black, stickered in sunshine* — thick 2–3px ink borders + hard offset shadows with no blur, that lift on hover and sink on press. Frame content as game **panels** and **quests**. Type is DM Sans (display) + Inter (body) + Press Start 2P (tiny pixel labels only) + Caveat (handwriting). See `README.md` for the full content + visual + iconography rules.
